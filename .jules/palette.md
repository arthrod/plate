## 2024-03-24 - Accessibility improvements for icon-only buttons\n**Learning:** Icon-only buttons using `size="icon"` often lack accessible names when implemented purely with visual icons (like `<Settings>` or `<XIcon>`). This pattern is pervasive across the components and templates folder. It creates unnavigable elements for screen reader users.\n**Action:** Always ensure that icon-only buttons include an `aria-label` attribute (or a nested `<span className="sr-only">`) to provide clear, actionable context for assistive technologies. Additionally, remember to sync these updates between `apps/www/src/registry/` and `templates/plate-playground-template/`.
